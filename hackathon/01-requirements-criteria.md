## Global Eligibility

The hackathon allows projects that are executed fully by AI agents, or through
AI tools built/manipulated by humans.

## Grand Champion

Highest honor, awarded across all tracks.

Scoring:

| Dimension | Weight | Description |
|---|---:|---|
| Technical Depth | 30% | AI x on-chain integration, architecture completeness, code quality |
| Innovation | 25% | Originality, whether it proposes a new AI x Web3 paradigm |
| Mantle Ecosystem Contribution | 25% | Substantive use of Mantle and long-term value to the ecosystem |
| Product Completeness | 20% | Runnable demo, UX, scalability |

Requirements:

- Must be deployed on Mantle Network.
- Submit open-source repo.
- Submit runnable demo.
- Submit project pitch.
- Must be nominated from at least one track.

## Agentic Economy Track

Exclusively supported by Byreal. This is the primary track for Agentic Wallet
Treasury.

Two winning paths:

1. **Human Driven — DeFi Deep Dive**: Use Byreal Agent Skills / Byreal Perps
   CLI / RealClaw to explore advanced on-chain trading strategies.
2. **AI Driven — RealClaw Real-Life Expansion**: Take RealClaw beyond DeFi and
   into real-world everyday use cases.

Core components:

| Component | Capability |
|---|---|
| Byreal Agent Skills | Calls Byreal CLMM, supports LP & Swap |
| Byreal Perps CLI | Calls Byreal Perps, executes perpetual futures trades |
| RealClaw | OpenClaw-based agent with Byreal Skills pre-installed, extensible to non-DeFi scenarios |

Scoring:

| Dimension | Weight | Description |
|---|---:|---|
| General | 70% | Byreal integration depth, agent autonomy, technical completeness, sustainability |
| Track-specific | 30% | DeFi strategy alpha OR RealClaw real-world validity |

Encouraged directions:

- News-driven trading agent using Byreal Agent Skills / Byreal Perps CLI.
- Market-making or arbitrage strategies.
- Automated portfolio rebalancing with on-chain execution.
- Personal CFO agent.
- On-chain life manager.
- Health data management.
- Everyday decision-making assistant.

Requirements:

- Must use at least one core capability: Byreal Agent Skills, Byreal Perps CLI,
  or RealClaw.
- Deploy on Mantle or Solana.
- Submit open-source repo.
- Submit runnable demo.
- Submit one-line pitch.

Submission question to answer:

> Which Byreal on-chain capabilities does your project use? What scenario are
> they applied to?

## 20 Project Deployment Award

First-come, first-served; 20 projects only. No judge scoring, but every bar
must be cleared.

Technical deployment:

- Smart contract deployed on Mantle Mainnet or Testnet.
- Contract is verified on Mantle Explorer.
- At least one AI-powered function is callable on-chain, such as agent trigger,
  inference result written on-chain, or automated execution.

Product completeness:

- Frontend demo is publicly accessible, not localhost.
- Deployment address included in DoraHacks submission.
- Demo video of at least 2 minutes walking through the core use case.

Documentation:

- Open-source GitHub repo.
- README with setup instructions.
- Architecture overview.
- Deployed contract address.

## Best UI/UX Award

Scoring:

| Dimension | Weight | Description |
|---|---:|---|
| Visual Design | 30% | Aesthetic quality, design consistency, brand identity |
| Interaction & Flow | 30% | Smooth interactions, user guidance, responsiveness |
| AI Interaction Design | 25% | Whether AI capabilities are presented naturally |
| Accessibility | 15% | Beginner-friendliness, lowering Web3 barrier |

Requirements:

- Runnable frontend interface.
- Demo video or publicly accessible link.

## Community Voting

The project with the most DoraHacks votes wins.

What wins:

- Clear, compelling demo for non-technical users.
- Real pain point.
- Strong community presence and shareability.

## Other Track Notes

Alpha & Data Track:

- Uses Mantle on-chain data as a core source.
- Must deploy on Mantle.
- Must explain data sources, AI role, and verifiable value.

AI & RWA Track:

- Must involve real-world assets.
- Must deploy on Mantle.
- Must explain asset type, AI role, and Mantle realization.
